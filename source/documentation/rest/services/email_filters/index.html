Email Filters API

Email Filters are a set of rules that are applied to triggering email's body, subject and from address. It only applies to generic_email kind of Services. The way multiple filters are combined depends on the email_filter_mode attribute of the service.
Resource Description
POST services/:service_id/email_filters Create a new Email Filter for the specified service.
PUT services/:service_id/email_filters/:id Update an existing Email Filter.
DELETE services/:service_id/email_filters/:id Delete an existing Email Filter.
