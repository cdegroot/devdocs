PUT services/:id/enable

Enable a previously disabled service.

Resource URL 

PUT https://<subdomain>.pagerduty.com/api/v1/services/:id/enable

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X PUT \
    "https://acme.pagerduty.com/api/v1/services/PIJ90N7/enable"

Note: Content-type of the request must be application/json

Response Status: 200 OK
