GET schedules/:id/users

List all the users on-call in a given schedule for a given time range.

Resource URL 

GET https://<subdomain>.pagerduty.com/api/v1/schedules/:id/users

Parameters

Name Type Required Description
since Date No The start of the date range over which you want to return on-call users.
until Date No The end time of the date range over which you want to return on-call users.

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    --data-urlencode "since=2012-08-01" \
    --data-urlencode "until=2012-09-01" \
    "https://acme.pagerduty.com/api/v1/schedules/PIJ90N7/users"

Note: Content-type of the request must be application/json

Response Status: 200 OK
