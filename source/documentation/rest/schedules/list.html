GET schedules

List existing on-call schedules.

Resource URL

GET https://<subdomain>.pagerduty.com/api/v1/schedules

Parameters

This action is paginated. See the pagination documentation for details.
NameTypeRequiredDescription
queryStringNoFilters the result, showing only the schedules whose name matches the query.
requester_idStringNoThe user id of the user making the request. This will be used to generate the calendar private urls. This is only needed if you are using token based authentication.

Example

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    "https://acme.pagerduty.com/api/v1/schedules"

Note: Content-type of the request must be application/json

Response Status: 200 OK
