Notification Rules API

When a user is assigned an incident, the user's notification rules are used to determine which contact method will be used to alert the user and how long after the incident to do so.

Access and manipulate the notification rules for a user.
Resource Description
GET users/:user_id/notification_rules List existing notification rules for the specified user.
GET users/:user_id/notification_rules/:id Get details for a notification rule.
POST users/:user_id/notification_rules Create a new notification rule for the specified user.
PUT users/:user_id/notification_rules/:id Update an existing notification rule.
DELETE users/:user_id/notification_rules/:id Remove a notification rule.
