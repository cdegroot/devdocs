GET users/:user_id/notification_rules

List existing notification rules for the specified user.

Resource URL 

GET https://<subdomain>.pagerduty.com/api/v1/users/:user_id/notification_rules

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    "https://acme.pagerduty.com/api/v1/users/PPSFHH7/notification_rules"

Note: Content-type of the request must be application/json

Response Status: 200 OK
