DELETE users/:user_id/notification_rules/:id

Remove a notification rule.

Resource URL 

DELETE https://<subdomain>.pagerduty.com/api/v1/users/:user_id/notification_rules/:id

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X DELETE \
    "https://acme.pagerduty.com/api/v1/users/PPSFHH7/notification_rules/PHU1XBC"

Note: Content-type of the request must be application/json

Response Status: 204 No Content
