GET users/:user_id/notification_rules/:id

Get details for a notification rule.

Resource URL 

GET https://<subdomain>.pagerduty.com/api/v1/users/:user_id/notification_rules/:id

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X GET -G \
    "https://acme.pagerduty.com/api/v1/users/PPSFHH7/notification_rules/PHU1XBC"

Note: Content-type of the request must be application/json

Response Status: 200 OK 

{
  "notification_rule": {
    "id": "PHU1XBC",
    "start_delay_in_minutes": 0,
    "contact_method": {
      "id": "PNGKILZ",
      "label": "Work",
      "type": "email",
      "email": "bob@acme.com",
      "address": "bob@acme.com",
      "send_short_email": false,
      "user_id": "PPSFHH7"
    }
  }
}
