DELETE users/:user_id/contact_methods/:id

Remove a contact method and any corresponding notification rules.

Resource URL 

DELETE https://<subdomain>.pagerduty.com/api/v1/users/:user_id/contact_methods/:id

Example 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X DELETE \
    "https://acme.pagerduty.com/api/v1/users/PPSFHH7/contact_methods/P6W4C77"

Note: Content-type of the request must be application/json

Response Status: 204 No Content
