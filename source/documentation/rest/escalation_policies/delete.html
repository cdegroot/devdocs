DELETE escalation_policies/:id

Deletes an existing escalation policy and rules. The escalation policy must not be in use by any services.

Resource URL 

DELETE https://<subdomain>.pagerduty.com/api/v1/escalation_policies/:id

Example

Delete an escalation policy 

curl -H "Content-type: application/json" -H "Authorization: Token token=E7px6VVr3PVHZPJq51oa" -X DELETE \
    "https://acme.pagerduty.com/api/v1/escalation_policies/PF106PL"

Note: Content-type of the request must be application/json

Response Status: 204 No Content
