Escalation Policies API

This API lets you access and manipulate escalation policies and rules.
Resource Description
GET escalation_policies List existing escalation policies.
POST escalation_policies Create a new escalation policy.
GET escalation_policies/:id Get details about an escalation policy.
PUT escalation_policies/:id Update an escalation policy.
DELETE escalation_policies/:id Delete an escalation policy.
