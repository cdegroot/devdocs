Escalation Rules API

This API lets you access and manipulate escalation rules of an existing escalation policy.
Resource Description
GET escalation_policies/:escalation_policy_id/escalation_rules List escalation rules for an escalation policy.
POST escalation_policies/:escalation_policy_id/escalation_rules Append a new escalation rule to the existing escalation rules.
PUT escalation_policies/:escalation_policy_id/escalation_rules Updates the entire collection of escalation rules.
GET escalation_policies/:escalation_policy_id/escalation_rules/:id Get details about an escalation rule.
PUT escalation_policies/:escalation_policy_id/escalation_rules/:id Update an escalation rule.
DELETE escalation_policies/:escalation_policy_id/escalation_rules/:id Delete an escalation rule.
